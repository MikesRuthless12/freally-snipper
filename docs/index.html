Freally Snipper 0.19.84 — Home window

Freally Snipper 0.19.84 adds the Windows-11-style home window — a New / Camera / Video / Snippet / Timer / Color toolbar, a recent-captures gallery, a full settings pane with an 18-language picker, an About panel, and an opt-in Print Screen takeover.

New Features

  • A Win11-style toolbar starts a capture in your chosen snippet mode after an optional timer.
  • A recent-captures strip shows thumbnails you can reopen with one click.
  • Settings now cover the hotkey, save folder, image format, theme, and an 18-language UI picker.
  • An About panel shows the version, ownership, and the full license and third-party notices.
  • Print Screen can open Freally Snipper (opt-in) and restores your Windows default when turned off.
