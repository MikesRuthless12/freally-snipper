Freally Snipper 0.75.0 — Video editor & self-updater

Phase 6 video editor; per-OS installers + self-updater; captions removed. Screen recording in the owned freally-video format — record a region, a window (and follow it), or the full screen, with system audio, microphone, and an optional webcam. Play it back in-app, and export to GIF, WebM, or MP4.

New Features

  • Record a region, a window (followed as it moves and resizes), or the full screen to a .fvid — at any resolution, including 4K — with a Pause / Resume / Stop bar.
  • Capture system audio (what you hear) and/or your microphone, and overlay your webcam as a picture-in-picture.
  • Play any recording back in-app (Play / Pause / Loop) through the owned decoder.
  • Export a recording to an animated GIF (built-in), or to WebM or MP4 via ffmpeg.
  • Per-OS installers (.msi and one-line install.ps1 / install.sh) with a built-in self-updater.
  • A draft End User License Agreement and Privacy Policy in the About panel — your content is yours, and nothing is collected.
0.45.0 Released Jun 17, 2026

Freally Snipper 0.45.0 — Image editor

A full WYSIWYG image editor: markup tools, movable text / shapes / images / colour emoji, live filters and transforms, and on-device OCR — plus start-at-login and an in-app model downloader with live progress.

New Features

  • Edit any capture on a zoom/pan canvas — pen, brush, highlighter, eraser, shapes, text, watermark, placed images, and colour emoji — all movable and resizable. Save writes exactly what you see.
  • Live filters (grayscale, sepia, blur, sharpen, contrast, cartoonize…) and transforms (rotate, flip, bevel, crop), plus an eyedropper.
  • Extract Text (OCR) copies the words out of an image — over a dragged region or the whole image.
  • Start Freally Snipper at sign-in (minimized to the tray), and a Models panel that downloads each optional model on demand with an exact-size progress bar.
0.30.0 Released Jun 17, 2026

Freally Snipper 0.30.0 — Capture overlay action bar

Freally Snipper 0.30.0 adds a top-center action bar to the capture overlay — switch the snippet shape on the fly, set the markup colour, and (with Markup on) open each snip in an editor preview right below your selection.

New Features

  • A top-center action bar lets you switch the snippet shape — rectangle, window, freeform, full screen — live, mid-capture.
  • Turn on Markup to open each snip in an editor preview centered below your selection, with Save and Discard.
  • The bar hides the moment you start dragging, so it is never in the shot.
  • Set the markup colour right on the overlay; the Video and Text Extractor (OCR) buttons preview what is coming next.

0.19.84 Released Jun 17, 2026

Freally Snipper 0.19.84 — Home window

Freally Snipper 0.19.84 adds the Windows-11-style home window — a New / Camera / Video / Snippet / Timer / Color toolbar, an on-screen capture timer, a recent-captures gallery, a system tray, a full settings pane, an About panel, and an opt-in Print Screen takeover.

New Features

  • A Win11-style toolbar starts a capture; with a timer you pick the region first, then a countdown runs and the live screen is grabbed.
  • A recent-captures strip shows dated thumbnails you can reopen with one click.
  • A system tray (Windows/macOS) keeps the hotkey working while the window is closed.
  • Settings cover the hotkey, save folder, format, theme, an 18-language picker, a capture-editor toggle, and minimize-to-tray.
  • An About panel shows the version, ownership, and the full license and third-party notices.
  • Print Screen can open Freally Snipper (opt-in) and restores your Windows default when turned off.
