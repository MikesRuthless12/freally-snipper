# Freally Snipper — owned icon set (starter / draft)

These SVGs are **original works © 2026 Mike Weaver — All Rights Reserved** (see `../../../LICENSE`),
the seed of Freally Snipper's **owned** icon set. They are transparent-background and will be
rendered in-app via `resvg`/`usvg` (Phase 4). Permissive icon sets (Phosphor/MIT, Lucide/ISC) are an
optional fallback only, behind an icon abstraction.

**Convention:** the writing tools (highlighter, pen, brush) deliberately **face/write to the RIGHT**
(nib at lower-right) so the set is unmistakably original — not the usual left-facing stock icons.

Starter set (drafts): colored tools `highlighter` (yellow), `pen`, `brush`, `eraser` (all right-facing);
plus line-style action glyphs `text`, `text-extract` (Text Extractor / OCR), `rectangle`, `oval`,
`line`, `arrow`, `crop`, `rotate`, `zoom-in`, `zoom-out`, `undo`, `redo`, `save`, `copy`, `trash`,
`filters`, `watermark`. Capture-overlay / home-toolbar / video-editor icons come next.

> **Draft:** these are a first pass, authored without a local SVG renderer to preview them. Open
> them in a browser/editor and tell me what to refine (proportions, colors, weight). Completing the
> full owned set is a post-1.0 goal (roadmap §7), like `freally-font`.
